Architecture

Deep Research Pipeline

Multi-stage pipeline that transforms a simple query into comprehensive, structured research results.

1

Query Expansion

Generate 3-5 orthogonal subqueries covering recent developments, tutorials, API docs, troubleshooting, and comparisons.

2

Parallel Search

Execute all subqueries across multiple search engines concurrently for maximum coverage.

3

Deduplication & Scoring

URL normalization + content hashing. Score by authority domain, content type, freshness, and position.

4

Multi-Pass Crawling

BFS with depth limit. Prioritize same-domain links. Optional external link following for deeper coverage.

5

Content Extraction

trafilatura for clean content + code-aware analysis (21 languages) + API signature extraction + package detection.

6

Synthesis & Formatting

Aggregate by topic with confidence scores. Format into token-efficient markdown with quality badges and relevance scores.

Code-Aware

Built for Coding Agents

Every fetched page is analyzed for coding-agent relevance with rich metadata.

Installation

Get Started in Seconds

Install clco-deep-research in your favorite coding agent.

JSON 
{
  "mcpServers": {
    "clco-deep-research": {
      "command": "python3",
      "args": ["-m", "clco_deep_research.server"]
    }
  }
}

Add to ~/.claude.json

JSON 
{
  "mcpServers": {
    "clco-deep-research": {
      "command": "python3",
      "args": ["-m", "clco_deep_research.server"]
    }
  }
}

Add to ~/Library/Application Support/Claude/claude_desktop_config.json (macOS)

JSON 
{
  "mcpServers": {
    "clco-deep-research": {
      "command": "python3",
      "args": ["-m", "clco_deep_research.server"]
    }
  }
}

Add to .cursor/mcp.json or Cursor Settings → MCP

JSON 
{
  "servers": {
    "clco-deep-research": {
      "type": "stdio",
      "command": "python3",
      "args": ["-m", "clco_deep_research.server"]
    }
  }
}
