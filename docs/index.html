maru-deep-pro-search는 상용 API 키 등록 없이 사용하는 개발자 지향적 딥리서치 MCP 서버입니다. 9개 검색 소스를 연동한 자동 페일오버, 로컬 Granite 임베딩 기반 시맨틱 재랭킹, 그리고 정확한 출처 인용(Citations) 파이프라인을 지원하여 에이전트가 정확한 최신 팩트를 기반으로 작업하도록 강제합니다. maru-deep-pro-search is a developer-centric, deep-research MCP server requiring zero API keys. Featuring 9-engine aggregated search with auto-failover, local semantic re-ranking with Granite 97M, and a strict citation pipeline, it forces coding agents to use fresh, verified facts instead of hallucinating.

동작 흐름Workflow

간단한 3단계 프로세스 Simple 3-Step Process

  1. 1

    설치 및 등록Install & Register

    설치 명령어를 터미널에서 실행하여 패키지를 내려받은 후, 개발 환경에 MCP 서버 설정을 적용하고 에이전트를 재시작합니다. Run the installation command in your terminal, configure the MCP server in your agent, and restart the application.

  2. 2

    지침 템플릿 등록Inject System Rules

    에이전트의 시스템 설정(User Rules 등)에 코드 생성 전 실시간 탐색과 정확한 인용을 지시하는 템플릿을 등록합니다. Add instructions to your agent's system prompt to enforce real-time web queries and citations before code blocks.

  3. 3

    개발 질문 제기Ask and Verify

    구현할 기능의 구체적인 명세, 패키지 버전 또는 발생한 에러 로그를 명시하면 관련 자료를 검색하여 출처와 함께 코드를 도출합니다. Specify library versions, API specifications, or error traces. The agent retrieves the documentation, cites sources, and writes accurate code.
