maru-search

API 키 없이 검색엔진을 직접 스크래핑하고 인출 기반 답변을 제공하는 범용 AI 검색 MCP 서버.

PyPI · GitHub · README

pip install maru-search

}

도구

도구용도
answer질문에 대한 인출이 포함된 직접 답변
web_search검색엔진 스크래핑, 순위 결과와 인출 ID 반환
search_with_citations학술/기술 문서 작성용 사전 번호 인출 검색
fetch_page단일 URL에서 깨끗한 콘텐츠 추출
fetch_bulk다중 URL 병렬 페치
deep_research쿼리 자동 확장, 상위 결과 크롤링, 인출 기반 종합
stealthy_fetch보호된 사이트용 전체 안티봇 우회
parallel_search다중 검색 동시 실행

도구 선택 기준

  • 빠른 답변이 필요한가? → answer
  • 출처가 필요한가? → web_search 또는 search_with_citations
  • URL이 이미 있는가? → fetch_page 또는 fetch_bulk
  • 차단되었는가? → fetch_page에 stealth=True, 그래도 안 되면 stealthy_fetch
  • 심층 분석이 필요한가? → deep_research

차별점

  • 유료 API 없음 — OpenAI, Google API, Bing API 없음. 오직 직접 스크래핑만.
  • 인출(Citation) — 모든 결과에 [1], [2] ID 부여. LLM이 정확히 출처를 인용할 수 있음.
  • 다중 엔진 — SearchEngineRegistry로 새 스크래퍼 추가가 간단함.
  • BM25 순위 — 로컬 관련성 점수 + 권위도/최신성 메타데이터.
  • 코드 인식 — 21개 언어 감지, API 시그니처 추출, 코드-텍스트 비율 측정.

MARU_SEARCH_RETRIES3재시도 횟수

아키텍처

src/maru_search/
├── server.py        # MCP 서버 (8개 도구, 3개 프롬프트)
├── config.py        # 환경 변수 기반 런타임 설정
├── tools.py         # 도구 구현 + 레지스트리
├── engines/
│   ├── registry.py  # SearchEngineRegistry (팩토리)
│   ├── base.py      # SearchEngine ABC
│   └── duckduckgo.py
├── research/
│   ├── deep.py      # 딥리서치 + 답변 합성
│   ├── ranker.py    # BM25 + 메타데이터 순위
│   └── expander.py  # 쿼리 확장
├── extraction/
│   ├── code.py      # 21개 언어 감지
│   └── content.py   # 토큰 인식 자르기
└── utils/
    ├── url.py       # URL 정규화 / 필터 / 중복 제거
    └── retry.py     # 지수 백오프
