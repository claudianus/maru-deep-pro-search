내장 웹 검색의 6가지 치명적 결함 6 Fatal Flaws of Built-in Web Search

편리해 보이지만, 실제로 의존하면 에이전트가 환각에 빠집니다 Convenient in theory. Dangerous in production.

단일 엔진 의존

Single Engine

DuckDuckGo가 차단되면 끝입니다. 폴오버가 없어서 하나의 장애가 전체 서비스를 마비시킵니다.

DuckDuckGo blocks you? Game over. No failover means one outage kills everything.

랭킹 부재

No Ranking

검색엔진이 뱉은 순서 그대로 전달합니다. BM25, 권위도, 신선도 기반 재랭킹이 전혀 없습니다.

Raw engine order. No BM25, no authority scoring, no freshness filtering.

인용 환각

Citation Hallucination

출처를 지어내거나 아예 무시합니다. `[1]`, `[2]` 네이티브 인용 아키텍처가 없어서 검증이 불가능합니다.

Makes up sources or ignores them. No native citation IDs means zero traceability.

얕은 콘텐츠

Shallow Fetch

스니펫만 긁어 끝냅니다. API 문서, 버전 표, 코드 예시를 놓치고 얕은 블로그 글에 의존합니다.

Grabs a snippet and stops. Misses API docs, version tables, code examples entirely.

방어 없음

Zero Defense

임의의 웹 페이지를 필터 없이 가져옵니다. 프롬프트 인젝션, 제로폭 문자, 악성 콘텐츠에 무방비입니다.

Fetches arbitrary pages with no sanitization. Prompt injection, zero-width chars, malicious content all pass through.

강제 불가

Not Enforced

"검색할 수 있다"일 뿐, 아무것도 강제하지 않습니다. 에이전트는 여전히 낡은 학습 데이터를 기본으로 사용합니다.

The agent *can* search, but nothing forces it. It still defaults to stale training data every single time.

킹갓슈프림 리서치 플랫폼 Supreme Research Platform

검색 도구가 아닙니다. 리서치를 강제 실행하는 플랫폼입니다. Not a search tool. A research enforcement platform.

7엔진 자동 폴오버

7-Engine Failover

DuckDuckGo · Bing · Google · Naver · Qwant · Startpage · SearXNG. 하나가 망가지면 다음이 즉시 이어받습니다. 장애에 강인합니다.

Seven engines with automatic failover. One goes down, the next picks up instantly. Resilient by design.

DuckDuckGo Bing Google Naver Qwant Startpage SearXNG

Perplexity급 랭킹

Perplexity-Grade Ranking

BM25 관련성 + 밀집 시맨틱 유사도 + 권위도/신선도/코드밀도 점수. 최고의 소스가 정상에 오릅니다.

BM25 relevance + dense semantic similarity + authority/freshness/code-density scoring. Best sources rise to the top.

네이티브 인용

Native Citations

모든 주장에 `[1]`, `[2]`, `[3]`이 붙습니다. 출처는 실재하며 추적 가능하고 응답에 직접 포함됩니다.

Every claim gets `[1]`, `[2]`, `[3]`. Sources are real, traceable, and embedded directly in responses.

딥 리서치 파이프라인

Deep Research Pipeline

쿼리 자동 확장 → 다각도 검색 → 안티봇 에스컬레이션 스마트 페치 → 갭 탐지 → 인용된 종합 답변.

Auto query expansion → multi-angle search → anti-bot smart fetch → gap detection → synthesized cited answer.

프롬프트 인젝션 방어

Prompt Injection Defense

가져온 콘텐츠를 정제합니다: 제로폭 문자 제거, 채팅 토큰 중화, 의심 패턴 플래깅. 외부 콘텐츠를 신뢰할 수 없는 입력으로 처리합니다.

Sanitizes fetched content: strips zero-width chars, neutralizes chat tokens, flags suspicious patterns. Treats all external content as untrusted.

리서치 우선 강제

Research-First Enforcement

설정 CLI가 에이전트에 강제 규칙을 주입합니다: "어떤 코드를 작성하기 전에 반드시 deep_research를 호출하라." 예외 없음.

Setup CLI injects mandatory rules: "You MUST call deep_research before writing ANY code." No exceptions.
