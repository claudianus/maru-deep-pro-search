기능

딥리서치에 필요한 모든 것

정보를 수집, 분석, 종합하기 위해 코딩 에이전트에 특화된 6가지 강력한 도구.

🔍

web_search

Scrapling으로 검색엔진 SERP를 직접 스크래핑. API 키 불필요. 결과마다 콘텐츠 유형 힌트와 권위 배지 제공.

4개 엔진 안티봇
📄

fetch_page

모든 URL에서 깨끗하고 LLM 최적화된 콘텐츠 추출. 네비게이션, 푸터, 광고 제거. 품질 신호가 포함된 구조화된 마크다운 반환.

trafilatura 마크다운
⚡

fetch_bulk

설정 가능한 동시성으로 여러 URL을 병렬로 가져오기. 각 결과는 LLM 우선순위 지정을 위한 품질 신호 포함.

병렬 5 동시
🧠

deep_research

전체 파이프라인: 쿼리 확장 → 병렬 검색 → 관련성 스코어링 → 다중 패스 크롤링 → 콘텐츠 종합.

쿼리 확장 BFS 크롤러
🥷

stealthy_fetch

Cloudflare Turnstile, DataDome 등의 완전한 안티봇 우회. 브라우저 지문을 자동으로 적응.

Cloudflare DataDome
🚀

parallel_search

여러 검색을 엔진 전체에서 병렬로 실행. 각 쿼리는 최대 커버리지를 위해 독립적으로 스크래핑.

다중 쿼리 Scatter-gather
아키텍처

딥리서치 파이프라인

간단한 쿼리를 포괄적이고 구조화된 연구 결과로 변환하는 다단계 파이프라인.

1

쿼리 확장

최신 개발, 튜토리얼, API 문서, 문제 해결, 비교를 커버하는 3-5개의 직교 하위 쿼리 생성.

2

병렬 검색

모든 하위 쿼리를 여러 검색엔진에서 동시에 실행하여 최대 커버리지 달성.

3

중복 제거 및 스코어링

URL 정규화 + 콘텐츠 해싱. 권위 도메인, 콘텐츠 유형, 최신성, 위치로 점수화.

4

다중 패스 크롤링

깊이 제한 BFS. 동일 도메인 링크 우선순위. 더 깊은 커버리지를 위한 선택적 외부 링크 추적.

5

콘텐츠 추출

깨끗한 콘텐츠를 위한 trafilatura + 코드 인식 분석(21개 언어) + API 시그니처 추출 + 패키지 감지.

6

종합 및 포맷팅

신뢰도 점수로 주제별 집계. 품질 배지와 관련성 점수가 포함된 토큰 효율적 마크다운으로 포맷.

$ /parallel_search ["python vs go concurrency", "python vs rust performance"]
