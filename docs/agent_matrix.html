Layer 1 — Server session marking

First call to answer, deep_research, or other research-producing tools marks the session. Receipts land in ~/.maru/receipts/. Tiered drift: lockfile-only = soft; pyproject.toml / package.json = hard.

Layer 2 — Client hooks

Four agents with physical or near-physical blocks: Aider, Hermes, Claude Code, Cursor (2026+). Others rely on protocol injection + Layer 1.

Layer 3 — generate_code

Requires matching research_id and citation [N] from the session research packet (from answer or deep_research).
